# IST105 Assignment 12 - Handling Exceptions

## Description
This project demonstrates exception handling and testing in Python.

## Features
- Input five integers
- Handle ValueError
- Handle ZeroDivisionError
- Generic exception handling
- Print debugging
- Even/Odd checking using bitwise
- Sorting numbers greater than 10

## Testing Scenarios

### Test 1: Correct Input
5, 10, 15, 20, 25

### Test 2: Non Integer Input
5, abc, 10, 20, 30

### Test 3: Zero Division
0, 0, 0, 0, 0
